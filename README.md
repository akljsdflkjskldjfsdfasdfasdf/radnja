# Radnja — platforma za vođenje prodavnice

Faza 1: praćenje rokova trajanja (prijem robe, "ističe uskoro", otpis).

## Pokretanje na laptopu

1. Otvori terminal u ovom folderu.
2. Pokreni: `venv\Scripts\python app.py`
3. Otvori u pretraživaču: http://localhost:5000

## Tehnologije

- Python + Flask (web aplikacija)
- SQLite (baza podataka — od Koraka 2)
- Hosting: Render (besplatno)
